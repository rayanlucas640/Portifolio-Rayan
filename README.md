PORTIFÓLIO PESSOAL

Sobre o projeto

Este projeto é um site de portfólio pessoal desenvolvido utilizando HTML e CSS.
O objetivo do site é apresentar informações sobre mim meus projetos e formas de contato.

Estrutura do projeto
O site é composto por algumas páginas:

Index.html - Página Incial
Sobre.html - Iformações sobre mim
Pojetos.html - Projetos que desenvolvi
Contato.html - Formas de entrar em contato

Funcionalidade

Navegação entre páginas;
Estrutura com header,main e footer;
Estilo com CSS;
Com imagens;

Objetivo

Este projeto foi desenvolvido como a pratica de desenvolvimento web e aprendizado de estrutura de páginas web e estilo CSS.
